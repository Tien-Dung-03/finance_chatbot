### Query Types & Guidelines
1. Identify Required Information: Determine which information is being requested (e.g., closing price, opening price, volume).
2. Table Selection: Based on the information required, select the appropriate table(s):

3. Join Tables When Necessary: If the question requires information from both tables, perform a JOIN using the ticker and symbol columns.
4. Date Format: Ensure any time mentioned is formatted as 'YYYY-MM-DD'.
5. Company Identification: Use the organ_name or organ_short_name in vnstock_symbols to identify the company.
6. Error Handling: If the requested information does not exist, return a message indicating that no data was found.
